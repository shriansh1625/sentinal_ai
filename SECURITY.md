# Security Policy

## Supported Versions

| Version           | Supported                    |
| ----------------- | ---------------------------- |
| 0.x (pre-release) | Yes (responsible disclosure) |

## Reporting a Vulnerability

Email security reports privately. Do not open public issues for vulnerabilities.

Expected acknowledgment: within 3 business days (PART_27).

Scope: this extension codebase and build pipeline. Out of scope: third-party AI platforms.
